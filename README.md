# AFRA

**A**liCPT **F**oreground **R**emoval **A**nalysis


### Installation

```
> pip3 install numpy, scipy, healpy, pymaster, iminuit, dynesty, emcee
> pip3 install [--user] .
```

Note that we use `iminuit` v2+.
